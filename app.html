Instructions

Inside the castle are 25 doors, many with icons on the front.

The first door is unlocked, this is where you start your adventure.

Each day, enter the most recently unlocked door, solve the puzzle and unlock the next door ready for the next day.

Each door holds a unique puzzle to solve which will tell you the door icon, the symbol which appears on the door, and the door code, the code used to unlock it, for the next day's door.

Inside each room, you will have to use logic and features of your mobile device. If you get stuck, click the ' question_mark ' button for hints. Click the ' contract ' button to read the narration.

For each floor of the castle, there are some stairs or an exit sign, which will move you between the floors.
